# Pokemon Battles (Still in progress, not yet completed)

## Description 
Its a turn-based pokemon battle game where you can choose one of three total starter pokemons that each have different stats and moves. The game has classic features like a health bar, critical hit chances and a bot you will battle.

## Features
- 3 different starter pokemons you can choose from with different stats 
- turn based battle system
- health bar
- printing letter by letter like in the original games
- Exception handling
- 5% critical hit chance each turn (double damage)

## Coding-Features
- Pokemons are created as class (object oriented programming)
- if conditions
- while loops
- for loops
- custom functions
